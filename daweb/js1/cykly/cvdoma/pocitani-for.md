Přepište všechny WHILE cykly z předchozího cvičení na FOR cykly.

:::solution

1. ```js
   console.log('První FOR cyklus');
   for (let i = 1; i <= 100; i++) {
     console.log(i);
   }
   ```
1. ```js
   console.log('Druhý FOR cyklus');
   for (let i = 2; i <= 100; i += 2) {
     console.log(i);
   }
   ```
1. ```js
   console.log('Třetí FOR cyklus');
   for (let i = 100; i >= 0; i -= 2) {
     console.log(i);
   }
   ```

:::
